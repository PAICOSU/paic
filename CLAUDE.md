# CLAUDE.md for the Precision Agriculture Innovation Center Website

This file provides instructions for Claude Code to build a clean, modern, and professional website for the Precision Agriculture Innovation Center.

## Project Goal

To create a visually appealing, easy-to-navigate, and informative static website that showcases the lab's research, team, publications, and teaching activities. The website should be built using HTML, CSS, and JavaScript, and it must be fully responsive.

## General Instructions

- **Tone and Style**: Maintain a clean, modern, and academic tone throughout the website. Text should be clear, concise, and professional.
- **Responsiveness**: Ensure the website is fully responsive and functions flawlessly on desktops, tablets, and mobile devices.
- **File Structure**: Organize the website logically with separate HTML files for each page (e.g., `index.html`, `people.html`, `research.html`, etc.) and place CSS, JavaScript, and image files in their respective subdirectories (`/css`, `/js`, `/images`).
